Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
дэдлок возникает, из-за того, что вы не закрываем канал, 
поэтому main горутина будет бесконечно ждать данные.

```
